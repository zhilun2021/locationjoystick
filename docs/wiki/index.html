locationjoystick

Spoof your GPS location on Android. Use a floating joystick, saved routes, or automatic roaming to move your location anywhere while your other apps run normally.

Uses Android's official mock location API. Enable Developer Options, select locationjoystick as your mock location provider, and every app on the device sees the location you choose.

Install

  1. Download the latest APK from GitHub Releases.
  2. Open the downloaded file. If prompted "Allow installs from unknown sources", enable it in Settings for your browser or file manager, then return and retry.
  3. You may see "App scan recommended" from Google Play Protect — this is a standard warning shown for any APK installed outside the Play Store. Tap Install without scanning or scan first; either works.

First-run setup

The onboarding flow runs once when you first open the app. It walks you through the three permissions locationjoystick needs to work.

  • Location permission

    Grant Precise location so the map can center on your position and route recording works correctly.

  • Overlay permission

    Grant Display over other apps to enable the floating joystick and widget. The app opens the system settings page directly — toggle locationjoystick in the list and return to the app.

  • Mock location provider

    Enable Developer Options on your device (Settings → About phone → tap Build number seven times), then go to Settings → Developer Options → Select mock location app and choose locationjoystick. Tap Check again in the app — it verifies the setting and advances automatically.

  • All done

    Onboarding completes and you land on the Map screen ready to start spoofing. You won't see this flow again unless you clear app data.

Start spoofing

Tap Map, tap the start button, then long-press the map to walk or teleport anywhere. Use the floating widget or joystick overlay to control your location while other apps are in the foreground.

Updating

Open the new APK — Android will prompt "Replace app?" and preserve your data.

Switching from the Play Store version

The Play Store and GitHub builds use different signing keys, so Android will block a direct update. Export your data first via Settings → Export, then uninstall the Play Store version, install the GitHub APK, and re-import.
