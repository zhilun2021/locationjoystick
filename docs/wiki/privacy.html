Privacy Policy

Last updated: 2026-05-31

App Data

locationjoystick stores all user data — routes, favorites, speed profiles, and settings — locally on your device. No app data is transmitted to any server.

Location Permission

Location access is used solely to centre the map on your current position and to record routes. Location data is never uploaded or shared.

Camera Permission

Camera access is used solely for scanning QR codes during device-to-device data transfer. No images are stored or transmitted.

Accessibility Service

locationjoystick includes an optional, off-by-default "Compass tracking" feature (Settings → Menus → Tap to Walk) that uses Android's Accessibility Service API to take a screenshot of the screen immediately before you tap a location, in order to detect the on-screen compass direction of the app you're using. This lets the tap be converted into a correctly-oriented location update instead of assuming the screen is always oriented north-up. The screenshot is processed entirely on your device to read the compass heading and is never stored or transmitted anywhere. This feature is not an accessibility tool for users with disabilities — it is disabled by default and only active if you explicitly enable it.
