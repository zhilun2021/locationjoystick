# ISSUES.md

This document references known issues for agents to pick them and iterate on

## Roaming UX

- Sliders are too broad, let's use number inputs instead.
- Speed profiles outlined styles are inverted. the selected button should have the outlined colored, the other should be black
- Drawer height is too small, we need to scroll to see the start button, it should all be visible.
- There should be a "preview" button next to the "start" button that draws the dynamic route on the map before the user validates the roaming, this makes it easier to iterate on

All of the above should apply to both the "roaming" in settings and in the drawer map view
